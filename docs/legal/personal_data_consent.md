## 1. Согласие

Регистрируясь в FPV Last Run и отмечая согласие на обработку персональных данных, пользователь дает оператору согласие на обработку персональных данных в целях работы приложения, аккаунта, прогресса и рейтинга.

## 2. Персональные данные

Согласие распространяется на обработку следующих данных:

- email;
- display name;
- данные аккаунта и авторизации;
- password hash;
- refresh token hash;
- факт подтверждения возраста 13+;
- игровой прогресс;
- номера завершенных миссий;
- счет и лучшие результаты;
- бонусы за точность полета и попадание в танк;
- общий счет и уровень игрока;
- данные рейтинга;
- сведения о принятии юридических документов: тип документа, версия, дата и время принятия.

Также на устройстве пользователя могут храниться локальные данные приложения: жизни и таймер восстановления, гостевой прогресс, локальные достижения, настройки языка и звука, access token и refresh token в защищенном хранилище устройства.

## 3. Цели обработки

Данные обрабатываются для:

- регистрации и входа в аккаунт;
- защиты аккаунта;
- сохранения и восстановления игрового прогресса;
- расчета счета, уровня и открытых миссий;
- отображения профиля и рейтинга;
- подтверждения принятия юридических документов;
- удаления аккаунта по запросу пользователя;
- технической поддержки по обращению пользователя.

## 4. Действия с данными

Оператор может выполнять с данными действия, необходимые для работы приложения: сбор, запись, систематизацию, накопление, хранение, уточнение, использование, передачу внутри backend-сервиса приложения, обезличивание, блокирование, удаление и уничтожение.

## 5. Хранение и безопасность

Пароли не хранятся в открытом виде. На сервере хранится password hash. Refresh tokens на сервере хранятся только в виде hash.

Данные хранятся до удаления аккаунта, отзыва согласия или пока они необходимы для работы приложения и соблюдения применимых требований.

## 6. Отзыв согласия

Пользователь может отозвать согласие:

- удалив аккаунт в приложении через настройки;
- направив запрос на email: anpilovdmitriy@yandex.ru.

После отзыва согласия использование функций, требующих аккаунта и обработки данных, может стать невозможным.

## 7. Удаление аккаунта

В приложении предусмотрено удаление аккаунта через настройки с подтверждением пароля. При удалении аккаунта удаляются аккаунт, refresh token hashes, профиль, прогресс и юридические подтверждения, если их хранение не требуется законом.
