## Удаление аккаунта в приложении

В текущей версии FPV Last Run удаление аккаунта реализовано в приложении:

1. Откройте настройки.
2. Выберите удаление аккаунта.
3. Введите пароль для подтверждения.
4. Подтвердите удаление.

После успешного удаления локальные токены авторизации удаляются с устройства.

## Удаление аккаунта по email

Если доступ к приложению или аккаунту невозможен, пользователь может направить запрос на удаление аккаунта на email:

anpilovdmitriy@yandex.ru

В запросе нужно указать email аккаунта FPV Last Run. Для защиты аккаунта разработчик может запросить дополнительное подтверждение принадлежности аккаунта.

## Какие данные удаляются

При удалении аккаунта backend удаляет:

- аккаунт пользователя;
- refresh token hashes;
- профиль;
- игровой прогресс;
- юридические подтверждения;
- связанные с аккаунтом данные рейтинга и профиля, где применимо.

Локальные данные на устройстве, такие как гостевой прогресс, жизни, достижения и настройки, могут сохраняться до удаления приложения или очистки данных приложения, если они не связаны с удаленным аккаунтом.

## Что может быть сохранено

Некоторые данные могут сохраняться, если это требуется применимым законодательством Российской Федерации, необходимо для безопасности сервиса, рассмотрения споров или защиты прав.
