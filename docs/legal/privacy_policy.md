## Какие данные обрабатываются

В текущей версии FPV Last Run обрабатываются следующие данные.

Аккаунт и авторизация:

- email;
- пароль при регистрации/входе, который передается на сервер для проверки и хранится только в виде password hash;
- access token и refresh token, которые используются для авторизации;
- refresh token на сервере хранится только в виде hash;
- флаг подтверждения возраста 13+ при регистрации;
- факт принятия условий использования и согласия на обработку персональных данных.

Профиль:

- display name;
- признак, менялось ли имя игрока;
- общий счет;
- уровень игрока;
- статус premium, если функция будет включена позже.

Игровой прогресс:

- номера завершенных миссий;
- лучший счет по миссии;
- бонус за точность полета;
- бонус за попадание в танк;
- дата завершения миссии;
- количество завершенных миссий;
- открытая миссия.

Турнирная таблица:

- display name;
- общий счет;
- уровень игрока;
- позиция в рейтинге, рассчитанная на основе игровых результатов.

Юридические подтверждения:

- тип принятого документа;
- версия документа;
- дата и время принятия.

Локальные данные на устройстве:

- текущие жизни и время восстановления жизни;
- прогресс гостя по миссиям 1-2;
- локальные достижения и дата их открытия;
- настройки звука и языка;
- локально сохраненные токены авторизации в защищенном хранилище устройства.

## Для чего используются данные

Данные используются для:

- создания и защиты аккаунта;
- входа в аккаунт и поддержания сессии;
- сохранения игрового прогресса;
- расчета счета, уровня и открытых миссий;
- отображения профиля и рейтинга;
- проверки принятия юридических документов;
- работы локальных игровых систем, включая жизни, гостевой прогресс, достижения и настройки.

## Пароли и токены

Пароли не хранятся в открытом виде. На сервере хранится только password hash.

Refresh tokens на сервере не хранятся в открытом виде. На сервере хранится только hash refresh token. На устройстве пользователя access token и refresh token сохраняются в защищенном хранилище, если оно доступно платформой.

## Передача данных третьим лицам

В текущей MVP/pre-release версии данные используются для работы приложения и backend-сервиса FPV Last Run. Данные рейтинга могут быть видны другим пользователям в виде display name, счета, уровня и позиции в рейтинге.

Если позже будут подключены реклама, аналитика, crash reporting, платежные сервисы или другие сторонние поставщики, этот документ будет обновлен до или во время включения таких сервисов.

## Хранение данных

Данные аккаунта, профиля, прогресса, refresh token hashes и юридические подтверждения хранятся до удаления аккаунта или до тех пор, пока они нужны для работы сервиса и соблюдения применимых требований.

Локальные данные на устройстве хранятся до удаления приложения, очистки данных приложения или удаления соответствующих данных через доступные функции приложения.

## Удаление аккаунта

В приложении предусмотрено удаление аккаунта через экран настроек. Для удаления требуется подтвердить пароль.

При удалении аккаунта удаляются:

- аккаунт;
- refresh token hashes;
- юридические подтверждения;
- профиль;
- игровой прогресс.

Некоторые данные могут сохраняться только если это требуется законом или необходимо для защиты прав и безопасности сервиса.

Пока внешний URL для удаления аккаунта отсутствует:

- External account deletion URL: URL to be added before publication.
- До публикации в Google Play необходимо создать публичную страницу удаления аккаунта.
