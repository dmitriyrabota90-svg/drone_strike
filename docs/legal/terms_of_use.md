## 1. Общие положения

Настоящие Условия регулируют использование мобильной игры FPV Last Run и связанных с ней backend-функций: аккаунта, профиля, прогресса, рейтинга и юридических подтверждений.

Если пользователь не согласен с Условиями, он не должен создавать аккаунт или использовать функции, требующие аккаунта.

## 2. Возраст

FPV Last Run предназначена для пользователей 13+. При регистрации пользователь подтверждает, что ему исполнилось не менее 13 лет.

## 3. Аккаунт

Для части функций может требоваться аккаунт. Пользователь отвечает за корректность email, безопасность пароля и действия, совершенные через его аккаунт.

Пользователь не должен:

- передавать аккаунт третьим лицам;
- использовать чужой аккаунт;
- пытаться обойти авторизацию;
- вмешиваться в работу backend-сервиса;
- использовать ошибки, модификации или автоматизацию для нечестного получения прогресса, счета или места в рейтинге.

## 4. Игровой прогресс и рейтинг

FPV Last Run сохраняет прогресс миссий, лучший счет, бонусы, общий счет, уровень игрока и данные рейтинга.

Рейтинг может показывать другим пользователям display name, счет, уровень и позицию. Разработчик может исправлять или удалять некорректные результаты, если они получены из-за ошибки, злоупотребления или обхода правил.

## 5. Гостевой режим

Некоторые функции могут быть доступны без аккаунта. Гостевой прогресс хранится локально на устройстве и может быть ограничен. Удаление приложения или очистка данных приложения может привести к потере гостевого прогресса.

## 6. Покупки, premium, реклама и аналитика

В текущей MVP/pre-release версии покупки, premium account, реклама, аналитика и crash reporting не подключены как активные функции, если иное явно не указано в приложении.

Эти функции планируются. Перед их включением условия, политика конфиденциальности и интерфейс приложения должны быть обновлены, чтобы пользователь видел актуальные правила и сведения.

## 7. Доступность сервиса

FPV Last Run предоставляется в текущем виде. Работа сервера, рейтинга, синхронизации прогресса и других онлайн-функций может временно прерываться из-за обслуживания, ошибок, ограничений сети или изменений проекта.
